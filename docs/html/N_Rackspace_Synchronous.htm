PageExtensions
Provides synchronous extention methods for an IPageT instance.
Remarks
For example, CloudNetworkService only provides asynchronous methods, such as ListNetworksAsync(Identifier, NullableInt32, CancellationToken). If you prefer to not use async/await or Tasks, then by adding a using statement for Rackspace.Synchronous, synchronous methods like ListNetworks(CloudNetworkService, Identifier, NullableInt32) may be used.
